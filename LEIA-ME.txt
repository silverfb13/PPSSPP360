INSTALAÇÃO RÁPIDA:
------------------
1. Copie toda a pasta para o Xbox 360 (RGH/JTAG)
2. Coloque o arquivo ppsspp.ini em game:\
3. Coloque sua ISO do PSP como game:\psp.iso
4. Execute o emulador

OTIMIZAÇÕES APLICADAS:
----------------------
✓ FPS limitado a 30 (mais estável que 60)
✓ CPU configurada para 222 MHz (velocidade nativa PSP)
✓ Gráficos otimizados (resolução nativa, sem filtros pesados)
✓ Áudio funcional com qualidade decente
✓ Vertex Cache ativado
✓ Buffered Rendering para melhor compatibilidade
✓ 2 threads de trabalho (otimizado para Xbox 360)
